## Keyword Clustering Strategy

### Cluster 1: Gift Box Builder
- **EN Hub:** `/en/custom-gift-box` → build your own gift box, custom gift box, personalized gift box
- **AR Hub:** `/ar/custom-gift-box` → بوكس هدايا مخصص, اصنع بوكسك, بوكس هدية فاخر

### Cluster 2: Personalized Mugs
- **EN Hub:** `/en/personalized-mug` → personalized mug, engraved mug, custom mug names date
- **AR Hub:** `/ar/personalized-mug` → مج مخصص, مج محفور عليه اسم, مج بالصور

### Cluster 3: Couple Gifts
- **EN Hub:** `/en/couple-gifts` → couple gifts, matching gifts, gifts for him and her
- **AR Hub:** `/ar/couple-gifts` → هدايا كابلز, هدايا متشابهة, هدايا له ولها

### Cluster 4: Occasion-Based
- **EN Hubs:** `/en/birthday-gifts`, `/en/anniversary-gifts`
- **AR Hubs:** `/ar/birthday-gifts`, `/ar/anniversary-gifts`

### Cluster 5: Gender-Based
- **EN:** `/en/products?gender=male`, `/en/products?gender=female`
- **AR:** `/ar/products?gender=male`, `/ar/products?gender=female`
